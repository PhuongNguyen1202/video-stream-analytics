### Cài đặt
Để tiến hành quá trình xử lý, bạn phải cài đặt một số gói phụ thuộc như sau:
 - Python
 - Tensorflow
 - OpenCV

- Bạn có thể cài đặt các gói phụ thuộc bằng cầu lệnh sau:

```bash
pip install -r requirements.txt
```
